3D Printable Spider Quad Robot Concept

As part of my CAD course, we were tasked with designing a dynamic system and animating it. I am quite interested in robotics and want to eventually be able to make a robot myself. As such I chose to make a concept design for a 3D printable Spider Quad Robot.

Design Parameters 🖼

There were a few design parameters that were set for this project. These are as follows:
  1. Needs to have at least 6 unique parts
  2. Needs to be animated and show the motion of the object
  3. Make use of Chebyshev Lambda linkage
The first two parameters were from the course's project outline. The other 3 were self-imposed. At the time, I had discovered the interesting mechanism of the Chebyshev Lambda 4-bar linkage and this seemed like the perfect place to use it.

Design Process 🖼

This QuadRobot concept will be simple 4-legged robot, that can be controlled using an Arduino Microcontroller. The legs are driven motors connected to a four-bar linkage, and more specifically the Chebyshev lambda linkage. The special property of this linkage is that it can take rotational motion and convert it into an approximate straight-line motion. This is perfect for this usecase.
